# FIGHTSTARS / Combats

## What
Gaming application with player profiles, battles, rankings.

## Stack
- Next.js 14 App Router
- TypeScript strict
- NeonDB (PostgreSQL)
- Vercel deployment
- Telegram Mini App

## Users
- Gamers
- Mobile-first (Telegram)
- Competitive players

## Core Features
- Player profiles
- Battle system
- Rankings/leaderboards
- Achievements
- Stats tracking

## Database
```
users, players, battles, achievements, rankings, stats
```

## UI Requirements
- Gaming aesthetic
- Bold colors, high contrast
- Micro-animations
- Achievement badges
- Progress bars
- Leaderboard tables

## Design Direction
- Dark theme base
- Neon accents (cyan, magenta, gold)
- Futuristic/cyberpunk vibe
- Smooth transitions
- Celebratory animations for wins
